gafrojs examples

Standalone HTML pages, each self-contained — drop one into an <iframe> from a slide or a blog post.

  • Primitive gallery — one of each CGA primitive (point, vector, line, plane, sphere, circle, point pair).
  • Intersections — draggable sphere intersecting a fixed plane and line, updating live.
  • Motor interpolation — screw-motion geodesic between two motors via exp(t·log(...)).
  • Motor interpolation — comparison — left vs right screw, double cover (negated M1), and decoupled lerp+slerp side-by-side.
  • Franka line tracking — Franka Emika IK chasing a draggable target line.
  • Inverse kinematics — single-arm IK to a random target pose via Problem + GaussNewton.
  • Point constraints — point-on-line / -plane / -sphere objectives, switchable live.
  • Multi-objective IK — pose + rest-posture + joint-limit objectives composed in one problem.
  • Generic robot FK — load a robot description into a System and drive per-joint sliders.
  • robot_interface sim — the sim-control core API (interface → RobotInterface → controller); needs a backend/config.
